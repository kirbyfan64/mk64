mk64
====

A simple 64-bit kernel written in `Myrddin <http://eigenstate.org/myrddin/>`_.

It's still a work-in-progress (short version: the keyboard inputs don't work)...but it's still neat.

Building
********

::
   
   $ ./configure
   $ make

Running
*******

::
   
   $ make run # run in QEMU
   $ bochs # run in Bochs
